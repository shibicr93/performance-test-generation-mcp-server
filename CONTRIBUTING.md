# Contributing to Performance Test Generation MCP Server

Thank you for your interest in improving this MCP server! This project follows

## üöÄ How to Contribute

1. **Fork & Clone:** Create your own branch from `main`.
2. **Standard:** Follow the [Google Java Style Guide](https://google.github.io).
3. **Tools:** When adding new `@Tool` definitions, ensure the `description` is optimized for LLM understanding.
4. **Testing:** All logic should be covered by JUnit 5 tests.
5. **PR:** Submit a Pull Request with a clear description of the performance testing tool or fix you've added.
